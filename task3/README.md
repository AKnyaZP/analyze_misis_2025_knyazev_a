# Task 3

## Запуск

```bash
python task.py
```

Скрипт автоматически читает два JSON файла из директории `../data/task3/`:
- `Ranzhirovka-A.json`
- `Ranzhirovka-B.json`

## Описание main функции

Функция `main(a_json: str, b_json: str) -> str` объединяет два ранжирования от разных экспертов в одно согласованное ранжирование:

### Алгоритм:
1. Парсит два JSON-файла с ранжированиями экспертов A и B
2. Строит матрицы отношений для каждого ранжирования
3. Находит конфликты (когда A говорит i<j, а B говорит j<i)
4. Группирует конфликтующие элементы в кластеры эквивалентности
5. Упорядочивает кластеры на основе согласованных отношений
6. Возвращает результирующее ранжирование в JSON-формате

Функция возвращает JSON-строку с объединённым ранжированием, где конфликтующие элементы помещаются в один кластер (массив).
