# Task 4

## Запуск

```bash
python task.py
```

Скрипт автоматически читает JSON файл из директории `../data/task4/data.json`.

## Описание main функции

Функция `main(s: str) -> Tuple[Any, Any, Any]` анализирует данные опроса экспертов и вычисляет различные метрики согласованности:

### Возвращаемые значения:
1. **Первый элемент** — ранжирование альтернатив по медиане Кемени
2. **Второй элемент** — коэффициент конкордации Кендалла (W)
3. **Третий элемент** — дополнительные метрики и анализ согласованности экспертов

### Алгоритм:
- Парсит JSON с данными опроса (матрицы парных сравнений от разных экспертов)
- Строит агрегированную матрицу для всех экспертов
- Вычисляет оптимальное ранжирование альтернатив методом Кемени
- Рассчитывает коэффициент конкордации Кендалла для оценки согласованности экспертов
- Выполняет статистический анализ согласованности мнений

Функция предназначена для анализа экспертных оценок и построения консолидированного мнения группы экспертов.
