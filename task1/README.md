# Task 1

## Запуск

```bash
python task.py
```

С опциональными параметрами:

```bash
python task.py --csv path/to/data.csv --root 1 --print-nodes
```

### Параметры:
- `--csv` — путь к CSV файлу с рёбрами дерева (по умолчанию `../data/task1/test.csv`)
- `--root` — корневой узел дерева (по умолчанию `"1"`)
- `--print-nodes` — вывести "OK" перед результатом

## Описание main функции

Функция `main(s: str, e: str)` принимает CSV-строку с рёбрами дерева и корневой узел, возвращает пять матриц отношений:

1. **r1** — прямые дочерние отношения (parent → child)
2. **r2** — обратные дочерние отношения (child → parent)
3. **r3** — отношения предок-потомок с расстоянием ≥2
4. **r4** — обратные отношения предок-потомок
5. **r5** — отношения между узлами с общим родителем (siblings)

Функция строит дерево из входных данных, начиная с корня, и вычисляет все указанные отношения в виде булевых матриц.
