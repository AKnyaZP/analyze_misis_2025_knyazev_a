# Task 0

## Запуск

```bash
python task.py
```

## Описание main функции

Функция `main(s: str) -> Tuple[int, int]` принимает строку `s` с данными в формате CSV (пары чисел, разделенные запятыми) и возвращает кортеж из двух целых чисел:
- Первое число — количество уникальных узлов в графе
- Второе число — количество рёбер в графе

Функция парсит входные данные, строит граф из пар узлов и подсчитывает количество уникальных узлов и рёбер.
