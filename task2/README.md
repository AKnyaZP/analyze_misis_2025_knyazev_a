# Task 2

## Запуск

```bash
python task.py
```

Скрипт читает CSV из `../data/task2/test.csv` и запрашивает идентификатор корневого узла.

## Описание функции

Функция `task(s: str, e: str) -> Tuple[float, float]` (и эквивалентная `main(s: str, e: str)`) вычисляет по методике Мишурова С.С.:

- **H(M,R)** — энтропия структуры графа (отношения r1..r5 между элементами из задания 1);
- **h(M,R)** — нормированная оценка структурной сложности (H / H_ref).

Аргументы:
- `s` — CSV-строка с рёбрами графа подчинённости (каждая строка: `parent,child`);
- `e` — идентификатор корневого узла.

Возвращает кортеж из двух float с округлением до одного знака после запятой: `(H, h)`.

Формулы: для каждого узла и типа отношения считаются исходящие связи l_ij, P = l_ij/(n−1), частичная энтропия −P log₂ P; H_ref = c·n·k, c = 1/(e·ln 2) ≈ 0.5307, k = 5.
