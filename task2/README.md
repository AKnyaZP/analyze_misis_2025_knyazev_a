# Task 2

## Запуск

```bash
python task.py
```

С опциональными параметрами:

```bash
python task.py --csv path/to/data.csv --root 1
```

### Параметры:
- `--csv` — путь к CSV файлу с рёбрами дерева (по умолчанию `../data/task2/test.csv`)
- `--root` — корневой узел дерева (по умолчанию `"1"`)

## Описание main функции

Функция `main(s: str, e: str) -> Tuple[float, float]` вычисляет метрики энтропии для дерева:

- **h** — абсолютная энтропия структуры дерева (округлённая до 1 знака после запятой)
- **hn** — нормализованная энтропия (h / href, где href — эталонная энтропия)

Функция анализирует структуру дерева через пять матриц отношений (как в task1) и вычисляет энтропию на основе распределения связей между узлами. Возвращает кортеж `(h, hn)`.
